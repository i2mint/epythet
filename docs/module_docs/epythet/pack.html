epythet.pack¶

Utils to package and publish.

The typical sequence of the methodic and paranoid could be something like this:

` python pack.py current-configs  # see what you got python pack.py increment-configs-version  # update (increment the version and write that in setup.cfg python pack.py current-configs-version  # see that it worked python pack.py current-configs  # ... if you really want to see the whole configs again (you're really paranoid) python pack.py run-setup  # see that it worked python pack.py twine-upload-dist  # publish # and then go check things work... `

If you’re crazy (or know what you’re doing) just do

` python pack.py go `
