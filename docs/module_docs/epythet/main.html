epythet.main¶

epythet.main.go(project_dir, output_dirname='module_docs', skip_existing=True, docsrc_dir=None)[source]¶

Create sphinx autodocs and table of contents for module defined by setup.cfg

Parameters

  • project_dir – Path to root project directory containing setup.cfg

  • output_dirname – directory name to be created under docsrc

  • skip_existing – existing docs will not be overwritten if True

  • docsrc_dir – path to sphinx docs source file
