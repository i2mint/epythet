epythet.docs_gen.call_make¶

epythet.docs_gen.call_make.make(project_dir, *make_args)[source]¶

Invokes the Sphinx docs makefile in project_dir/docsrc. Commonly used make_args: “html”, “doctest”, “github”

Parameters

  • make_args – Sphinx docs makefile args
